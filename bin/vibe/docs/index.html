Vibe

Extract design tokens from any website. Scrapes with Puppeteer, analyzes colors, fonts, and spacing, then outputs CSS variables, Tailwind config, or JSON.

  • GitHub

Features

  • Puppeteer Scraping

    Launches a headless browser, navigates to any URL, and captures the full computed style tree.

  • Color Clustering

    Groups extracted colors by similarity, identifies primary, secondary, and accent palettes automatically.

  • Dark Mode Detection

    Detects prefers-color-scheme media queries and extracts both light and dark token sets.

  • Typography Extraction

    Pulls font families, weights, sizes, and line heights from headings, body text, and UI elements.

  • Multiple Output Formats

    Exports as CSS custom properties, Tailwind config, or raw JSON. Pipe into any design workflow.

Architecture

src/
  index.ts          CLI entry point, Commander.js setup
  scraper.ts        Puppeteer page navigation + style capture
  analyzer.ts       Color clustering, typography grouping, spacing analysis
  formatter.ts      Output rendering (CSS vars, Tailwind, JSON)

__tests__/
  scraper.test.ts   Scraping integration tests
  analyzer.test.ts  Token analysis unit tests
  formatter.test.ts Output format verification
